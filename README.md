# Quiz App (Flutter)

A simple quiz application built with Flutter as part of my learning journey.

This project focuses on understanding Flutter fundamentals such as UI layout, navigation, state handling, and user interaction. **This app is designed primarily for mobile devices.**

---

## About the Project

The app allows users to:

- Answer multiple-choice questions  
- Move between questions (Next / Back)  
- Skip questions and return later  
- Track quiz progress  
- View the final score  
- Restart the quiz  

This project was created to practice Flutter concepts and build a clear understanding of basic app structure and logic.

---

## Features

- Multiple-choice quiz interface  
- Next, Back, and Skip navigation  
- Progress tracking  
- Final results screen  
- Restart quiz option  
- Smooth page transitions  



## Tech Stack

- Flutter  
- Dart  
- Material Design  

## Prerequisites

Make sure you have installed:

- Flutter SDK  
- Android Studio or VS Code  
- Android emulator or physical device  

## How to Run

1. Clone the repository - git clone [https://github.com/mahanthvamsi/Quiz_app.git](https://github.com/mahanthvamsi/Quiz_app.git)

2. Navigate to the project folder -
   ```bash
   cd Quiz_app
   ```

6. Install dependencies -
   ```bash
   flutter pub get
   ```

8. Run the app -
   ```bash
   flutter run
   ```
